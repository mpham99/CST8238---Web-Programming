(i) What are different parts of a CSS style rule?

They are rule set, declaration, selector, property and value.

(ii) What are an element selector and grouped element selector in CSS? Provide an example of each.

Element selecter selects all instances of a given HTML element (Ex: p{} )
Grouped element selector selects and targets multiple elements with different name (Ex: p,h1,h2,h3 {})

(iii) What are class selectors in CSS? What are id selectors in CSS?

Class selector allows you to simutaneously target different HTML elements regardless of their position in the document tree.
id selector allows you to target a specific element by its id attribute regardless of its type or position.

(iv) What are contextual selectors in CSS?

Contextual selector allows you to select elements based on their ancestors, descendants or siblings.

(v) What are the three cascade principles used by browsers when style rules conflict?

They are inheritance, specificity, and location.
